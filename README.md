# DeepRacer Teleoperation

An open-source iOS/iPadOS application that lets you control your AWS DeepRacer vehicle like an RC car.

## Features

- Live video feed from your DeepRacer's camera
- Real-time remote control via intuitive touch interface
- Direct connection over your local network

## Prerequisites

- iOS/iPadOS device (iPhone or iPad)
- AWS DeepRacer vehicle
- Both devices connected to the same Wi-Fi network

## Setup Instructions

### 1. Connect DeepRacer to Wi-Fi

First, ensure your DeepRacer is connected to your local network. If you're setting this up for the first time, follow Amazon's guide:

[DeepRacer Wi-Fi Connection Guide](https://docs.aws.amazon.com/deepracer/latest/developerguide/deepracer-troubleshooting-wifi-connection-first-time.html)

### 2. Configure the App

1. Open the app on your iOS device
2. Enter your DeepRacer's IP address
3. Enter the device password
4. Tap on the device to connect

### 3. Start Driving

Once connected, you'll see the live video feed from your DeepRacer's camera and can control the vehicle using the on-screen controls.
